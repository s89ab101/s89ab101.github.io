# 記帳本 · 台灣版 PWA

## 安裝到 iOS 主畫面（像 App 一樣使用）

1. 用 **Safari** 開啟 `index.html`（或部署後的網址）
2. 點右下角「**分享**」按鈕（方框加箭頭圖示）
3. 向下滑，選「**加入主畫面**」
4. 點「新增」完成

安裝後圖示會出現在主畫面，全螢幕執行，支援**離線使用**。

---

## 功能清單

- ✅ 新增支出 / 收入，含8種支出分類
- ✅ 月份切換，查看歷史月份
- ✅ 本月收支摘要（支出 / 收入 / 結餘）
- ✅ 依日期分組的交易列表
- ✅ 近6個月支出長條圖
- ✅ 本月分類圓餅圖
- ✅ 各分類預算追蹤（超支警示）
- ✅ CSV 匯出功能
- ✅ 左右滑動切換頁籤
- ✅ 深色模式自動切換
- ✅ 離線可用（Service Worker）
- ✅ 資料儲存於裝置本機（localStorage）

## 部署

只需將整個資料夾上傳至任何靜態網站主機，例如：
- GitHub Pages
- Netlify
- Vercel

確保使用 HTTPS，Service Worker 才能正常運作。
